🌱 SoilProfiles.js - Tooltip & Metadata System

Interactive demonstrations of the tooltip and event callback features

1. Native SVG Tooltips (Default)

Mode: 'native' (default)
Behavior: Standard SVG <title> elements. Hover over horizons to see browser tooltips.
Use case: Quick, minimal tooltips without custom styling.
renderStaticToDOM(el, collection, { mode: 'depth', width: 400, height: 300, tooltips: { mode: 'native' } // or omit for default });

2. Custom Tooltips with Event Handlers

Mode: 'custom'
Behavior: Suppresses native tooltips, fires callbacks on hover/click with rich payload.
Use case: Rich, styled tooltips; detailed analytics; complex interactions.
Last Event:
(hover/click a horizon to see event details)
renderStaticToDOM(el, collection, { mode: 'depth', width: 400, height: 300, tooltips: { mode: 'custom' }, onHorizonHover: (payload) => { console.log('Hovered:', payload.horizon.name); }, onHorizonClick: (payload) => { console.log('Clicked:', payload.horizon.name); } });

3. Custom Tooltip Rendering

Mode: 'custom'
Behavior: Uses custom tooltipRenderer function to render styled HTML tooltips.
Use case: Branded tooltips; rich HTML content; complex layouts.
const customRenderer = (horizon) => { const div = document.createElement('div'); div.className = 'custom-tooltip'; div.innerHTML = ` <div>${horizon.name}</div> <div>${horizon.top}-${horizon.bottom}cm deep</div> `; return div; }; renderStaticToDOM(el, collection, { tooltips: { mode: 'custom' }, tooltipRenderer: customRenderer });

4. Data-Only Mode (No Native Tooltips)

Mode: 'data-only'
Behavior: Suppresses native <title> elements. Horizon data available via data attributes.
Use case: Advanced consumers who parse data attributes programmatically.
(Click button to inspect)
renderStaticToDOM(el, collection, { mode: 'depth', tooltips: { mode: 'data-only' } });

5. Comparison View with Tooltips

Mode: 'custom'
Behavior: Side-by-side profiles with event callbacks on each horizon.
Use case: Comparing multiple soil profiles.
renderComparison(el, collection, { mode: 'depth', height: 400, tooltips: { mode: 'custom' }, onHorizonClick: (payload) => { console.log(`Clicked horizon in ${payload.profileId}`); } });

6. Utility Functions

Available utilities:
  • formatHorizonProperties() - Format properties with custom labels/formatters
  • createDefaultTooltip() - Create styled HTML tooltip
  • getHorizonDataFromElement() - Parse data from SVG element
  • createTooltipLines() - Create lines for legacy tooltip system
(Click button to see utility output)
import { formatHorizonProperties, createDefaultTooltip } from 'soilprofiles'; // Format properties const props = formatHorizonProperties(horizon, { properties: ['name', 'depth', 'clay', 'ph'], customLabels: { clay: 'Clay %' }, customFormatters: { ph: (v) => v.toFixed(1) } }); // Create tooltip const tooltip = createDefaultTooltip(horizon, { className: 'my-tooltip', properties: ['name', 'depth', 'texture'] }); document.body.appendChild(tooltip);
